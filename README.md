<h2>📚 Contexto do projeto</h2>

<p>
Este projeto foi desenvolvido durante meu processo de aprendizado em desenvolvimento front-end.
</p>

<p>
A versão original foi construída com <strong>HTML, CSS e JavaScript puro</strong>, acompanhando um tutorial em vídeo. 
Posteriormente, realizei uma <strong>refatoração completa para React</strong>, focando em:
</p>

<ul>
  <li>Componentização</li>
  <li>Reutilização de código</li>
  <li>Organização de arquitetura</li>
  <li>Separação de responsabilidades</li>
</ul>

<p>
Essa evolução demonstra minha capacidade de modernizar aplicações e aplicar boas práticas do ecossistema React.
</p>

<hr />

<h2>🧠 Principais aprendizados</h2>

<ul>
  <li>Conversão de projetos estáticos para SPA</li>
  <li>Uso de React Hooks (<code>useState</code>, <code>useEffect</code>)</li>
  <li>Consumo de APIs REST</li>
  <li>Gerenciamento de estado</li>
  <li>Componentização</li>
</ul>

<h2>⚙️ Funcionalidades</h2>

<h2>📦 Como executar</h2>

<pre>
git clone https://github.com/joseisraelpiresduarte-dev/Pokedex_React.js.git
cd Pokedex_React.js
npm install
npm start
</pre>

<p>
Acesse em: <strong>http://localhost:3000</strong>
</p>
